# ⚡ Electricity Consumption & Bill Predictor

This project is a machine learning-based web application that predicts electricity consumption and estimates the electricity bill.

## 🔍 Features
- Predicts electricity usage using Linear Regression
- Calculates bill using slab-based pricing system
- Provides detailed analysis including:
  - Usage category (Low / Moderate / High)
  - Consumption insights
  - Energy-saving tips
- Interactive graph to visualize electricity cost vs units

## 🛠️ Technologies Used
- Python
- Streamlit
- Scikit-learn
- Plotly

## ▶️ How to Run
1. Install required libraries:
   pip install -r requirements.txt
2. Run the application:
   streamlit run app.py
